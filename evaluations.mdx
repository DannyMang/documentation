## Quick start

```python
from veri_sdk import Client

client = Client()

eval = client.evals.create(
    name="math accuracy",
    dataset_id="ds_abc123",
    scorers=[{"type": "exact_match", "name": "em", "reference_field": "label"}],
)

run = client.evals.runs.create(eval.id, model=f"deployment:{dep_id}", num_samples=200)
run = run.wait()

print(f"Mean score: {run.results['scores']['em']['mean']:.3f}")
```

## What's shipped (Phase A)

- `exact_match` scorer — case-insensitive by default, strips whitespace, coerces non-string references.
- `contains` scorer — substring or regex match against a dataset reference field.
- Model target: `deployment:<id>` against a SERVING deployment.
- Dataset support: JSONL uploads.
- Full run lifecycle with per-sample item rows + aggregate `mean/median/std/min/max/count/error_count` per scorer.
