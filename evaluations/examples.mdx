Practical recipes with the [Evaluations](/evaluations) SDK. Each assumes
`client = Client()` and an eval dataset with a `label` column.

## Run an eval and read the score

```python
eval = client.evals.create(
    name="gsm8k-exact",
    dataset_id="ds_abc123",
    scorers=[{"type": "exact_match", "name": "em", "reference_field": "label"}],
)

run = client.evals.runs.create(eval.id, model="dep_abc").wait()
print(run.results["scores"]["em"]["mean"])   # aggregate accuracy
```

## Inspect per-item failures

```python
run = client.evals.runs.create(eval.id, model="dep_abc").wait()

failures = [
    item for item in client.evals.runs.items(eval.id, run.id, limit=200)
    if item.scores.get("em", 0) < 1.0
]
for item in failures:
    print(f"[{item.item_index}] {item.input!r} -> {item.output!r} (want {item.label!r})")
```

## Compare two models against the same eval

Reuse one eval definition, run it against each target, and compare aggregates:

```python
eval = client.evals.create(
    name="compare-em",
    dataset_id="ds_abc123",
    scorers=[{"type": "exact_match", "name": "em", "reference_field": "label"}],
)

for target in ["Qwen/Qwen2.5-0.5B-Instruct", "dep_my_trained"]:
    run = client.evals.runs.create(eval.id, model=target).wait()
    print(target, run.results["scores"]["em"]["mean"])
```

## Multiple scorers at once

```python
eval = client.evals.create(
    name="multi",
    dataset_id="ds_abc123",
    scorers=[
        {"type": "exact_match", "name": "em", "reference_field": "label"},
        {"type": "contains", "name": "has_answer", "reference_field": "label"},
    ],
)
run = client.evals.runs.create(eval.id, model="dep_abc").wait()
print(run.results["scores"])   # {"em": {...}, "has_answer": {...}}
```
