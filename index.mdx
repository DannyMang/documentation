## Quick Example

<CodeGroup>

```python Python
from veri import VeriClient

client = VeriClient(api_key="your-api-key")
result = client.detect(open("image.jpg", "rb"))

print(f"Is AI-generated: {result.is_fake}")
print(f"Confidence: {result.confidence:.1%}")
```

```typescript TypeScript
import { VeriClient } from '@veri/sdk';

const client = new VeriClient({ apiKey: 'your-api-key' });
const result = await client.detect(imageBuffer);

console.log(`Is AI-generated: ${result.isFake}`);
console.log(`Confidence: ${(result.confidence * 100).toFixed(1)}%`);
```

```bash cURL
curl -X POST https://api.tryveri.com/v1/detect \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"image": "base64-encoded-image"}'
```
