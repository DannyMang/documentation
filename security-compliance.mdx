## Overview

Security and compliance are active workstreams for Veri, but they are not fully completed yet.

## SOC 2 Status

Veri is currently in the SOC 2 process.

That means:

- we are building the controls and documentation needed for the audit path
- we are not yet presenting Veri as SOC 2 certified
- this page will evolve as the program becomes more formalized

## What We Can Say Today

Based on the current product and specs:

- user resources are scoped by ownership checks
- billing and credit flows are tracked through Stripe
- storage and checkpoint handling are explicit in the control plane
- security hardening remains an active roadmap item

## Still In Progress

Important items still called out in the roadmap include:

- signed auth instead of trusted user headers
- tighter control-plane network boundaries
- encryption and secret-handling improvements
- broader production hardening before launch
