---
title: "Get Batch Results"
api: "GET /v1/batch/{id}/results"
description: "Get the results of a completed batch detection job"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The batch job ID.
</ParamField>

## Response

<ResponseField name="jobId" type="string">
  Unique batch job ID.
</ResponseField>

<ResponseField name="status" type="object">
  Job status object (same as GET /v1/batch/{id}).
</ResponseField>

<ResponseField name="results" type="object">
  Detection results keyed by image index (0-based).

  Each result contains:
  - `id` - Detection ID
  - `isFake` - Whether image is AI-generated
  - `confidence` - Confidence score (0-1)
  - `modelScores` - Individual model scores
  - `error` - Error message (if detection failed)
</ResponseField>

<RequestExample>

```bash cURL
curl https://api.tryveri.com/v1/batch/batch_xyz789abc/results \
  -H "X-API-Key: your-api-key"
```

```python Python
from veri import VeriClient

client = VeriClient(api_key="your-api-key")

results = client.get_batch_results("batch_xyz789abc")

for index, detection in results.results.items():
    if "error" in detection:
        print(f"Image {index}: Failed - {detection['error']}")
    else:
        print(f"Image {index}: {'Fake' if detection['isFake'] else 'Real'} ({detection['confidence']:.1%})")
```

```typescript TypeScript
import { VeriClient } from '@veri/sdk';

const client = new VeriClient({ apiKey: 'your-api-key' });

const results = await client.getBatchResults('batch_xyz789abc');

for (const [index, detection] of Object.entries(results.results)) {
  if (detection.error) {
    console.log(`Image ${index}: Failed - ${detection.error}`);
  } else {
    console.log(`Image ${index}: ${detection.isFake ? 'Fake' : 'Real'} (${(detection.confidence * 100).toFixed(1)}%)`);
  }
}
```

## Notes

- Results are only available after job status is `completed`
- Failed detections include an `error` field instead of detection results
- Results are retained for 24 hours after job completion
