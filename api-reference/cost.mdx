## Spend Summary

Get an overview of attributed job spend for the last `N` days.

<ParamField query="days" type="integer" default="30">
  Reporting window in days.
</ParamField>

### Request

```bash
curl "http://localhost:8000/v1/cost/summary?days=30" \
  -H "Authorization: Bearer $VERI_INTERNAL_API_KEY" \
  -H "X-Veri-User-Id: $VERI_USER_ID"
```

### Response

```json
{
  "period_days": 30,
  "job_count": 12,
  "total_raw_cost_usd": 142.5,
  "total_user_cost_usd": 185.25,
  "gross_margin_usd": 42.75,
  "gross_margin_pct": 23.1,
  "total_gpu_hours": 47.5,
  "avg_cost_per_job_usd": 11.875
}
```

## Estimate Job Cost

Estimate the cost of a run from GPU shape, duration, provider, and markup.

<ParamField query="gpu_type" type="string" default="A100-80GB">
  GPU type to price.
</ParamField>

<ParamField query="gpu_count" type="integer" default="1">
  Number of GPUs.
</ParamField>

<ParamField query="duration_minutes" type="number" default="60">
  Runtime used for the estimate.
</ParamField>

<ParamField query="provider" type="string" default="modal">
  Provider key from the pricing table.
</ParamField>

<ParamField query="markup" type="number" default="1.3">
  User-facing markup multiplier.
</ParamField>

### Request

```bash
curl "http://localhost:8000/v1/cost/estimate?gpu_type=A100-80GB&gpu_count=1&duration_minutes=60&provider=modal&markup=1.3" \
  -H "Authorization: Bearer $VERI_INTERNAL_API_KEY" \
  -H "X-Veri-User-Id: $VERI_USER_ID"
```

### Response

```json
{
  "raw_cost_usd": 3.472,
  "user_cost_usd": 4.5136,
  "gpu_cost_usd": 3.4,
  "storage_cost_usd": 0.0,
  "overhead_cost_usd": 0.072,
  "duration_seconds": 3600.0,
  "gpu_type": "A100-80GB",
  "gpu_count": 1,
  "provider": "modal",
  "markup_multiplier": 1.3
}
```
