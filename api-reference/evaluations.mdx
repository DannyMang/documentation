## What the API will provide

- Create and manage **Eval** definitions (a dataset plus one or more scorers).
- Start **Eval Runs** that score a model target and report progress.
- Read back aggregate statistics per scorer and per-sample outputs once a run completes.

Endpoint paths, request and response shapes, and scorer configuration will be added here when evaluations are released.
