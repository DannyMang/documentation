---
title: "Billing"
description: "Check credit balance and transaction history"
---

Veri uses a credit-based billing system backed by Stripe. Credits are purchased through the dashboard and deducted automatically when training jobs complete.

## Get Credit Balance

<ParamField query="stripe_customer_id" type="string" required>
  Your Stripe customer ID (linked to your account).
</ParamField>

### Request

```bash
curl "https://api.veri.studio/v1/billing/balance?stripe_customer_id=cus_abc123" \
  -H "Authorization: Bearer vk_your_api_key"
```

### Response

```json
{
  "stripe_customer_id": "cus_abc123",
  "balance_usd": 47.50
}
```

## List Transactions

<ParamField query="stripe_customer_id" type="string" required>
  Your Stripe customer ID.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Maximum number of transactions to return.
</ParamField>

### Request

```bash
curl "https://api.veri.studio/v1/billing/transactions?stripe_customer_id=cus_abc123&limit=10" \
  -H "Authorization: Bearer vk_your_api_key"
```

### Response

```json
{
  "items": [
    {
      "id": "txn_xyz789",
      "amount_usd": -4.51,
      "description": "Training job job_xyz789",
      "created_at": "2026-05-01T14:30:00Z"
    },
    {
      "id": "txn_abc456",
      "amount_usd": 50.00,
      "description": "Credit purchase",
      "created_at": "2026-04-28T09:00:00Z"
    }
  ]
}
```

## Check Balance

Check whether a user has sufficient credits for a job before submission. Returns 402 when the balance is insufficient.

<ParamField query="stripe_customer_id" type="string" required>
  Your Stripe customer ID.
</ParamField>

<ParamField query="estimated_cost_usd" type="number" required>
  Estimated cost of the job in USD.
</ParamField>

### Request

```bash
curl -X POST "https://api.veri.studio/v1/billing/check-balance?stripe_customer_id=cus_abc123&estimated_cost_usd=10.00" \
  -H "Authorization: Bearer vk_your_api_key"
```

### Response (sufficient)

```json
{
  "sufficient": true,
  "balance_usd": 47.50,
  "estimated_cost_usd": 10.00
}
```

### Response (insufficient)

```json
{
  "detail": "Insufficient credit balance"
}
```
