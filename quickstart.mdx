## Prerequisites

Before you begin, you'll need:
- A Veri API key (create one from the [dashboard](https://veri.studio/dashboard) or via the [API keys endpoint](/api-reference/api-keys))
- Python 3.10+
- A dataset, either as JSONL or from a supported external source
- A Python reward function in `trl` or `miles` format (for GRPO jobs)

## Recommended First Run

1. Sign in through the web app.
2. Create an API key from the settings page.
3. Upload or connect a dataset.
4. Upload a reward function (for GRPO) or skip this step (for video gen SFT).
5. Submit a fine-tuning job.
6. Watch the job status and cost update as the run progresses.

## API Walkthrough

### Set your API key

```bash
export VERI_API_KEY="vk_your_api_key"
```

### Upload a JSONL dataset

Each row must include a `prompt` field. The value can be plain text or a chat-style message array.

```json
{"prompt": "What is 15 * 23?"}
{"prompt": "Solve for x: 2x + 5 = 17", "expected_answer": "6"}
```

Upload it:

```bash
curl -X POST "https://api.veri.studio/v1/datasets?name=math_prompts" \
  -H "Authorization: Bearer $VERI_API_KEY" \
  -F "file=@math_prompts.jsonl"
```

### Or connect an external dataset

For example, a Hugging Face dataset:

```bash
curl -X POST https://api.veri.studio/v1/datasets/connect \
  -H "Authorization: Bearer $VERI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "gsm8k-train",
    "source_type": "hf",
    "hf_dataset": "gsm8k",
    "hf_config": {
      "split": "train",
      "column_mapping": {
        "question": "prompt",
        "answer": "label"
      }
    }
  }'
```

You can validate a connection before creating it:

```bash
curl -X POST https://api.veri.studio/v1/datasets/connect/validate \
  -H "Authorization: Bearer $VERI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "source_type": "s3",
    "source_uri": "s3://my-bucket/prompts.jsonl"
  }'
```

### Upload a reward function

TRL-format reward functions are the default:

```python
def reward(completions, answer=None, **kwargs):
    scores = []
    for completion in completions:
        if answer and str(answer) in completion:
            scores.append(1.0)
        elif "<answer>" in completion and "</answer>" in completion:
            scores.append(0.5)
        else:
            scores.append(0.0)
    return scores
```

Upload it:

```bash
curl -X POST https://api.veri.studio/v1/reward_functions \
  -H "Authorization: Bearer $VERI_API_KEY" \
  -F "name=math_reward" \
  -F "format=trl" \
  -F "file=@reward.py"
```

Use `format=miles` if your file defines `async def reward(args, sample, **kwargs) -> float`.

### Submit a training job

```bash
curl -X POST https://api.veri.studio/v1/training_jobs \
  -H "Authorization: Bearer $VERI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "base_model": "Qwen/Qwen3-4B",
    "dataset_id": "ds_abc123",
    "reward_function_id": "rf_def456",
    "output_name": "qwen3-4b-math",
    "hyperparameters": {
      "learning_rate": 1e-6,
      "num_epochs": 1,
      "rollouts_per_prompt": 8,
      "max_response_length": 2048,
      "global_batch_size": 64
    },
    "gpu": {
      "gpu_type": "A100-80GB",
      "gpu_count": 1
    },
    "checkpoint_destination": {
      "type": "veri"
    }
  }'
```

### Monitor the job

Poll status:

```bash
curl https://api.veri.studio/v1/training_jobs/job_xyz789 \
  -H "Authorization: Bearer $VERI_API_KEY"
```

Stream logs:

```bash
curl -N https://api.veri.studio/v1/training_jobs/job_xyz789/logs \
  -H "Authorization: Bearer $VERI_API_KEY"
```

### Download the final checkpoint

When the job reaches `completed`, fetch the download URL:

```bash
curl https://api.veri.studio/v1/training_jobs/job_xyz789/model \
  -H "Authorization: Bearer $VERI_API_KEY"
```
