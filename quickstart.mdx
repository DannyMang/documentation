This page proves the platform works end-to-end before you spend a cent on GPUs. Total time: about 3 minutes.

By the end of it you'll have:
- The `veri` CLI installed locally.
- An authenticated session.
- A successful `client.billing.balance()` round-trip against `https://api.veri.studio`.

When you're ready to spend real GPU time, jump to one of the three workflow quickstarts at the bottom of this page.

## Install

The CLI ships the Python SDK and CLI together.

<CodeGroup>
```bash uv (recommended)
uv tool install veri-sdk
```

```bash pipx
pipx install veri-sdk
```

```bash pip
pip install veri-sdk
```
</CodeGroup>

Verify:

```bash
veri version
```

<Note>
  The CLI uses `~/.config/veri/config.toml` on Linux/macOS (or `$XDG_CONFIG_HOME/veri/config.toml`) for credentials and defaults.
</Note>

## Log in

Create an API key from the [Veri dashboard](https://veri.studio/dashboard/settings), then:

<CodeGroup>
```bash Browser flow (default)
veri login
```

```bash Paste an existing key
veri login --paste
```

```bash Non-interactive
veri login --token vk_your_api_key
```
</CodeGroup>

Confirm:

```bash
veri whoami
```

Output:

```text
user_id:    usr_abc123
api_url:    https://api.veri.studio
config:     /Users/you/.config/veri/config.toml
```

<Tip>
  Prefer environment variables? Set `VERI_API_KEY=vk_...` and any CLI call will pick it up. Env vars take precedence over the config file.
</Tip>

## Make your first call

The cheapest possible round-trip — read your credit balance. No GPU work, no inference cost.

<CodeGroup>
```python Python SDK
from veri_sdk import Client

client = Client()

balance = client.billing.balance()
print(f"Available credit: ${balance.available_usd:.2f}")
```

```bash curl
curl https://api.veri.studio/v1/billing/balance \
  -H "Authorization: Bearer vk_your_api_key"
```
</CodeGroup>

Now ask the cost API what a training job would cost — still no spend, just a quote.

```python
estimate = client.cost.estimate(
    gpu_type="A100-80GB",
    gpu_count=1,
    hours=0.25,
)
print(f"Estimated 15-min A100 run: ${estimate.total_usd:.2f}")
```

If both calls succeed, the platform is healthy and your account is wired up.

## Troubleshooting

<AccordionGroup>
  <Accordion title="veri login can't open a browser">
    Use `veri login --paste` (paste a key you create in the dashboard) or `veri login --token vk_...` for fully non-interactive flows.
  </Accordion>

  <Accordion title="401 Unauthorized">
    Either the key was revoked from the dashboard, or `VERI_API_KEY` in your shell points at a stale value. Run `veri whoami` to see which key the CLI is using, and `veri logout` then `veri login` to refresh.
  </Accordion>

  <Accordion title="Connection errors / DNS failures">
    Confirm `https://api.veri.studio` is reachable from your network. Veri does not run a self-hosted control plane — every CLI call hits the production API.
  </Accordion>
</AccordionGroup>
